Description

Insert an empty row in the worksheet

Syntax

BOOL InsertRow( int nPos, BOOL bUndo = FALSE, int nCount = 1 )

Parameters

nPos
[input] Row number to insert before (zero offset)
bUndo
[input] Whether to support Undo (TRUE) or not (FALSE).
Its default value is FALSE.
nCount
[input] Number of rows to insert.

Return

TRUE on success, otherwise FALSE

Examples

EX1 

// First worksheet in project should have 12 rows of data to see effect
int Worksheet_InsertRow_Ex1()
{
    WorksheetPage wp = Project.WorksheetPages(0);
    if(!wp)
        return -1;
    
    Worksheet wks(wp.GetName());
    return (int)wks.InsertRow(10);
}

Remark

Insert an empty row in the worksheet
