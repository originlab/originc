Description

return string in the form of [Data1]Sheet1

Syntax

string make_book_sheet_name( const string & strBook, const string & strSheet )

Parameters

strBook
[input] book short name
strSheet
[input] sheet name

Return

[strBook]strSheet

Examples

EX1

void make_book_sheet_name_ex1()
{
    Worksheet wks;
    string strBookName  = "Book1";
    string strSheetName = "Sheet1";
    string strBookSheetName = make_book_sheet_name(strBookName,strSheetName );
    if(!wks_from_book_sheet_name(wks, strBookSheetName))
        out_str("No such worksheet exists");
}
