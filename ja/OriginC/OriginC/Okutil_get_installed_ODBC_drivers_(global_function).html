概要

インストールされているODBCドライバの名前を含むStringArray（またはvector<string>）を生成します。検出されたドライバの数を返します。

構文

int okutil_get_installed_ODBC_drivers(StringArray* psaDrivers);

パラメータ

psaDrivers
[output]ドライバ名を入力するStringArray（またはvector<string>）へのポインタ


戻り値

検出されたドライバの数

サンプル

EX1 

#include <Origin.h>
#include <outils.h>

void dump_drivers()
{
	vector<string> vs;
	int num = okutil_get_installed_ODBC_drivers(&vs);
	for (int i = 0; i < num; i++)
	{
		printf("%s\n", vs[i]);
	}
}
