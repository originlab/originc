Description

Get the state of progress dialog

Syntax

BOOL IsAbort( )

Parameters

Return

TRUE if clicked Cancel button, FALSE if didn't clicked.

Examples

EX1 

void progressBox_IsAbort_ex1()
{
    progressBox *pProgress = new progressBox("Please click cancel button to break.", PBOX_TOPMOST);
    while ( !pProgress->IsAbort() )
    {
    };
    delete pProgress;
}
